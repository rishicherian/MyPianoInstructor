A full-stack iOS ear training application built with **SwiftUI** and **Firebase**. 

## Features
- **Adaptive Ear Training:** "Listen Mode" challenges users to identify notes and chords by ear.
- **Accuracy Game:** Guitar Hero-style falling note game to test reflex and sight-reading.
- **Global Leaderboards:** Real-time high score tracking using **Cloud Firestore**.
- **Personalized Accounts:** Users can set custom usernames for the global rankings.

## Tech Stack
- **Frontend:** SwiftUI, AVFoundation (Audio Engine)
- **Backend:** Firebase Firestore (NoSQL Database)
- **Architecture:** MVVM (Model-View-ViewModel)

## Getting Started
1. Clone the repository.
2. Open `MyPianoInstructor.xcodeproj` in Xcode.
3. Add your own `GoogleService-Info.plist` from Firebase to the root folder.
4. Run on iPhone Simulator.
